about my journey

I have been into web design since 2004. I started in high school (dial-up, Notepad, almost no knowledge source), then at university, I created our department website. While other faculty web pages looked like a Word document, ours had pictures, CSS styles and some Java script. Then I tried WordPress, Joomla and Bitrix.

I had some experience creating rules for an AI moderation system that declined inappropriate reviews on Yandex (a Russian search engine). I used a lot of Excel, tried Tableau visualizations and IBM Statistics for data analysis. Later, I wrote about blockchain projects for ICOBazaar.com, preparing a very thorough analysis of different startups' investment potential. I studied some Ruby and worked on an Arduino project to create a Teddy bear singing with my voice.

I have also completed the Stanford Code in Place Python course, spend two quatrimesters studying Math and Computer Science in local community college. In my Math class we covered limits and derivatives and how these can be applied in economics, physics, chemistry, aerodynamics, and tried my first object-oriented language, Java, in my CS class. It turned out that for a Java beginner, knowing Python helps a lot, and the basic concepts are the same.

What I love about programming is the constant process of learning new and exciting things. After learning English, Spanish, Basque, French and Chinese, I can see the whole system, which helps a lot when trying new technologies.
